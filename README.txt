Pour utiliser le logiciel:
1) Utiliser le script calibrage dans le dossier calibrage. Celui-ci va générer des fichiers csv.
2) Modifier le code de flask_srv pour mettre les bons noms et bonnes concentrations pour que le calibrage soit pris en compte.
3) Appeler le script calibration pour visualiser la droite de calibration.
4) Lancer le serveur.
